# Introduction

This project is a Fitness applications that tracks user's run history.

### Technologies/Frameworks Used
1. Java 21
2. Spring Boot 3.2.3
3. Maven
4. Postgres
5. Docker
6. RESTful API
7. Postman

### IDE Used
* IntelliJ

### Getting Started
To run the application, please install the following:

* JDK 21
* Docker Desktop

### Coding Architecture
* Package by Feature
